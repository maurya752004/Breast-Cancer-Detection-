# Breast Cancer Detection Using Machine Learning

Breast cancer is one of the most dangerous diseases affecting women worldwide.  
If it is not identified at an early stage, it can lead to serious complications and even death.

According to global studies, nearly **12% of women worldwide are affected by breast cancer**, and the number continues to increase every year. Early detection plays a crucial role in increasing survival rates.
